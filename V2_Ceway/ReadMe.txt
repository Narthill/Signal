Run
输入三张图片，输出z、u、v轴调整量以及运行时间
同时绘画出三张图同一刃边位置的大致拟合情况

1、Select三张图片，自小向大依次添加
   图片应命名为“0.45.bmp”、“0.50.bmp”
   即图片名称位数应相同！
2、点击“Run”，运行
3、点击“Clear”，清空程序
********************************************


********************************************
Test
输入多张图片，输出六个刃边位置最清晰的图片名称
同时绘出多张图同一刃边位置的laplace值变化情况

1、Img_Num选择test的图片张数
2、Select自小向大依次添加
   图片应命名为“0.45.bmp”、“0.50.bmp”
   抑或“0.583.bmp”、“0.590.bmp”
   即图片名称位数应相同！
3、点击“Test”，运行
4、点击“Clear”，清空程序
